# Modern File Converter - Replit Project

## Overview
A modern web-based file converter with a beautiful UI supporting multiple file format categories (Audio, Image, Video, Document). This is a frontend demo application that simulates file conversion.

**Current State**: Fully functional static web application running on Python HTTP server

## Project Structure
- `index.html` - Main application file (HTML/CSS/JavaScript)
- `server.py` - Python HTTP server configured for Replit (port 5000, no caching)
- `README.md` - Project documentation

## Technologies
- **Frontend**: Pure HTML5, CSS3, JavaScript (ES6+) - no external dependencies
- **Server**: Python 3.11 HTTP server
- **Deployment**: Static site served on port 5000

## Important Notes
- This is a **demo application** - file conversion is simulated on the frontend
- For actual file conversion, backend processing would be needed (FFmpeg, ImageMagick, or conversion API)
- Server configured with cache control headers to prevent caching issues in Replit iframe environment
